### 작성한 코드   

```cs
// 이미지 속 강아지 코드를 작성해본다.

class Program
{
    static void Main(string[] args)
    {   
        // 왕왕     
        Console.WriteLine("|\\_/|");
        Console.WriteLine("|q p|   /}");
        Console.WriteLine("( 0 )\"\"\"\\");
        Console.WriteLine("|\"^\"`    |");
        Console.WriteLine("||_/=\\\\__|");        

    }    
    
}
```
